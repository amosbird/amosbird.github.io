Windows window manager?

2016-05-19

What is the most ergonomic way to jump to the target window on Windows? This question bothers me long before I start pursuing ergonomic research.
I’ve tried so many tools, the most recent one is GoToWindow. They all make me feel cumbersome. Until I wrote my customized ahk script, I finally got the feeling of ergonomic flow.

The concept of my ergo switching is, one comb-key for a specific window. For instance, ‘C-M-e’ => emacs, ‘C-M-f’ => firefox etc. I’ve never keeped more than
ten applications at one time, so remembering these key bindings aren’t a big deal. This switching strategy delivers instant responding time which is really smooth.

Here is what I’ve done:

  1. Write a AHK script to bind windows with key combinations.

  2. Make all apps visible on the task bar.

Please give it a try. It’s very satisfying. If you have any question, feel free to leave a comment. Any suggestions are very much appreciated.
